# next 项目搭建过程

## 项目搭建过程

1. 生成文件 .gitignore
2. npm init
3. sudo npm install --save next react react-dom
4. 新建 /pages/index.js

   ```js
   export default () => <div>Welcome to next.js!</div>;
   ```

5. npm run dev

   也可以运行 npm run dev -- -p <端口号>

6. 静态目录 static
